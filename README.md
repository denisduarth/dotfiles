# Configuração para Linux do dotfile

Passos para instalar os arquivos nas pastas de script (arquivo de instalação dos pacotes do S.O e configuração de arquivos como .zshrc e .zsh_aliases).

    git clone https://github.com/denisduarth/dotfiles.git
    cd dotfiles
    ./install.sh (para instalação do script em ~/dotfiles/scripts/bootstrap.sh)
